# DFX API Wrapper

A TypeScript tool that interacts with the public DFX Swiss API to display supported crypto assets grouped by blockchain.

## What it does

- Fetches all supported assets from the DFX Swiss API
- Groups assets by blockchain
- Displays a clean overview of DFX's supported ecosystem

## Tech stack

- TypeScript
- Node.js
- Axios

## How to run

Clone the repository:
```bash
git clone https://github.com/Zmooth-Operator/dfx-api-wrapper.git
```

Install dependencies:
```bash
npm install
```

Run the wrapper:
```bash
npx tsx index.ts
```

## About

Built to explore the DFX Swiss public API. DFX is a Swiss crypto on/off-ramp supporting 80+ assets across 20+ blockchains.
